# Pathange Naga Lakshmi — Portfolio

Personal portfolio for **Pathange Naga Lakshmi**, Senior UX/UI Designer.
Static HTML/CSS/JS — no build step. Deploys directly to Vercel.

## Structure

```
.
├── index.html            # Home / landing
├── case-goodfit.html     # Case study — GoodFit
├── case-hirelens.html    # Case study — HireLens
├── tweaks-panel.jsx      # In-page tweak controls (design tooling)
├── tweaks.jsx            # Tweak definitions
├── assets/               # Images, covers, portrait
└── vercel.json           # Hosting config
```

## Run locally

Just open `index.html` in a browser, or serve the folder:

```bash
npx serve .
```

## Deploy

This is a static site. Push to GitHub, import the repo into Vercel, and Vercel will detect "Other / Static" and deploy as-is. No build command required.
